# typescript-project
This is a sample TypeScript project.


# Setup

Install node and npm.

Check version

```shell
node --version
npm --version
```

Install typescript globally via npm.

```shell
npm install -g typescript
```

Now clone this project.

```shell
git clone https://github.com/yusufshakeel/typescript-project.git
```

Install packages.

```shell
npm i
```
